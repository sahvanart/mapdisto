About

MapDisto is a program for mapping genetic markers in experimental segregating populations like backcross, F2, doubled haploids, single-seed descent, highly recombinant lines. Read more about Mapdisto in his dedicated blog article.

New

Programs, tools, etc

If you need these programs, please check the downloads page or send us an email.
These tools need Microsoft Excel 2007 or later.

CSSL Finder

is a program for managing introgression lines, and particularly useful to develop Chromosome Segment Substitution Lines (CSSLs). Thanks to its graphical interface, CSSL Finder is very easy to use. It can:

  • Search a population of advanced backcrossed lines for a set of lines that optimizes the representation of the donor genome
  • Display graphical genotypes of the lines
  • Display graphical phenotypic values of the lines
Download

cM Converter

is a program (centimorgan converter) to convert in batch physical to genetical map positions of molecular markers of rice. It uses the in silico mapping data provided by the Rice Genome Annotation project.

Download

DarwinXporter

is a tool to export diversity analysis data for Darwin from Excel workbooks (data sheets).

Download

SNP Extractor

is designed to extract optimal subsets of SNP markers from large lists of SNPs, based on several criteria like chromosomal position or highest probability of polymorphism.

NOISYmputer

We present NOISYmputer, program for imputation of LC-NGS data that eliminates the need of complex pre-filtering of noisy data, accurately types heterozygous chromosomic regions, corrects erroneous data, and imputes missing data. Read more.

Paddy Map

is a database for easily generating sets of polymorphic SSR markers in inters-pecific and intra-specific crosses of rice (Oryza sativa L.). It is based on the concept of Universal Core Genetic Map that we are developing in the framework of a Generation Challenge Program project. Version 1 of the Universal Core Genetic Map for rice is referenced in Gramene as “CIAT SSR 2006”. Version 2, which contains several corrections for marker identities and locations, is available from the author and is included in Paddy Map. Paddy Map was mainly developed by Matthieu Bouniol.

Paddy Genes Book

is a database of phenotypes observed in a collection of T-DNA insertional rice mutants.

LSD Analyzer

allows identifying traces of ancient or recent common ancestry in rice, through a graphical analysis of Large SNP Data (LSD).
