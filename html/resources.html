Resources

Click on the links below to download resources.

  • DB_Rice_SSRs_v_5.1.zip
    Use this file with the "Extract map from DB" function in MapDisto to get the map positions of non-duplicated SSRs in the rice genome.
  • mapmaker-OSX-Intel.zip
    This is the Mapmaker installer for Mac OS X machines (Intel only). Note: I've noticed that this version of Mapmaker may not be able to find intitial subsets of markers when using the 'Order all sequences' command. Double click on installer, then put the "mapmaker" folder in the "Applications" folder. The embedded "mmk.command" file is for MapDisto for Excel 2011.
  • mapmaker-OSX-BDS-Intel.zip
    This is the Mapmaker installer for Mac OS X machines (Intel only, big dataset). Note: I've noticed that this version of Mapmaker may not be able to find intitial subsets of markers when using the 'Order all sequences' command. Double click on installer, then put the "mapmaker" folder in the "Applications" folder. The embedded "mmk.command" file is for MapDisto for Excel 2011.
  • mapmaker-OSX-PPC.zip
    This is the Mapmaker installer for Mac OS X machines (Power PC only). Note: I've noticed that this version of Mapmaker may not be able to find intitial subsets of markers when using the 'Order all sequences' command. Double click on installer, then put the "mapmaker" folder in the "Applications" folder. The embedded "mmk.command" file is for MapDisto for Excel 2004.
  • mapmaker-DOS.zip
    This is the Mapmaker installer for DOS-compatible PCs. It will often crash (I've only tested under Windows XP). Several re-lauchings of the Mapmaker commands from MapDisto may be necessary.
  • mmk.command XL2011.zip
    Replace the "mmk.command" file by this file in the "mapmaker" install folder if you use Excel 2011 for Mac OS X.
  • mmk.command XL2004.zip
    Replace the "mmk.command" file by this file in the "mapmaker" install folder if you use Excel 2004 for Mac OS X.

Author

Mathias Lorieux.

I'm a researcher of the Institut de Recherche pour le Développement (IRD), a French institute dedicated to scientific research for the developing countries, where I’m part of the RICE group a team of the DIADE Research Unit.

I also work with CIAT, Cali, Colombia, where I'm leading the Rice Genetics and Genomics group, a joint research laboratory between IRD and CIAT.
